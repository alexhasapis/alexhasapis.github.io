NY High Schools DB

NY High Schools DB is a project taking multiple seperate datasets involving New York City public high schools taken from NYC Open Data and putting them together in an interactive site. The idea began with my passion for youth services. I was interested in how different schools served the youth of NYC. Each school has charts for different performance metrics as well as shows surrounding after-school programs available using the Google Maps API.

The site is built using a Ruby on Rails server, a MySQL database. In addition to the Google Maps API, the front uses D3.js to render graphic data on performance. The site is styled using Foundation 5.

Warblr

Warblr is collaborative project as part of the General Assembly Labs Hackathon. The project involved Web Development students, UX students and Project Management Students. Groups worked together to create a concept and app over a one week sprint and presented to a panel of industry professionals. It was a great learning experience working in a team environement similar to the tech industry!

Our team used Rails and Javascript to create the site. Upon selection of music input choices the browser sent an AJAX call to the server which in turn made an API call to the Echonest API to create a unique playlist for the user. A Spotify music player was then created so that a user could listen to the playlist.

Golden Ticket Diner

This project is a mockup restaurant POS system for maintainance for ordering food at the diner. The site has CRUD functionality in that users could add, view, delete and update food items, parties and orders. When a party is closed a receipt is created showing the items ordered, quantity, grand total and suggested gratuity.

I used Sinatra and RESTful routes to build the site. Items, orders and parties were stored in a PostgreSQL server using ActiveRecord.
